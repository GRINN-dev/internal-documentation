# Introduction

Bienvenue sur le site de documentation interne de Grinn. Vous y trouverez deux rubriques principales:

- tech: une documentation techniques sur les technologies et choix techniques de l'agence
- process: une documentation sur les process de gestion mis en place pour assurer le suivi des projets et leur achevement dans les temps impartis suivant le standard de qualite de l'agence

## C'est pour qui ?

C'est pour tout le monde ! (pas que les devs !). Pour les nouveaux, vous vous y ferez une idee concrete de la maniere dont nous travaillons et apprendrez les bases de notre stack techniques rapidement pour la mettre en pratique dans votre premier projet. Pour les vieux, c'est l'occasion de mieux maitriser des points techniques sur lesquels vous etes moins a l'aise, ou encore c'est un moyen puissant d'uniformiser et harmoniser nos production pour qu'on ne puisse plus dire: "untel fait comme ca mais pas moi"

## Qui contribue ?

Votre CTO prefere en priorite, mais il saura certainement vous mettre a contribution. Oh, et les contributions sont les bienvenues sous forme de pull request !
